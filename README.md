# blog-ska

Ska blog...


Create new posts:
* Band:
    ```
    hugo.exe new --type band posts/<BAND>
    ```
    where `<BAND>` is the directory name that will contain the new post.
* Default/Basic
    ```
    hugo.exe new --type posts posts/<POST>
    ```
    where `<POST>` is the directory name that will contain the new post.
